# Custom Decision Tree & Random Forest Implementation
### Machine Learning - Assignment 3

This project features a modular, from-scratch implementation of the **CART** algorithm and **Random Forest** ensembles, applied to the **Adult Income Dataset**.

## Project Structure

The codebase is organized into three specialized modules:

1.  **`decision_tree.py`**: Contains the core `Node` class and the `DecisionTree` classifier. It implements recursive binary splitting using **Gini Impurity** and **Shannon Entropy**.
2.  **`random_forest.py`**: Implements the `RandomForest` ensemble class. Key features include **Bootstrap Aggregating (Bagging)** and **Feature Subsampling** to ensure tree diversity.
3.  **`experiments.py`**: The main execution script. It handles:
    * Data preprocessing and cleaning of the Adult Income dataset.
    * Model training and benchmarking against `scikit-learn`.
    * Hyperparameter tuning (Grid Search).
    * Generation of all 8+ performance figures used in the report.

Theoretical Background
The implementation is built on the following mathematical principles to determine the optimal split at each node:

Shannon Entropy: Measures the impurity or disorder within the target labels.

Information Gain: The reduction in entropy achieved by partitioning the data on a specific feature.

Gini Impurity: Measures the probability of misclassification for a randomly chosen element.

Bootstrap Aggregating (Bagging): Reduces variance by training multiple trees on random subsets of the data.

Summary of Findings

Feature importance analysis revealed that Age, Capital-Gain, and Education-Num are the primary predictors of income level.

Modularizing the code into decision_tree.py and random_forest.py allowed for cleaner implementation of the bagging and recursive splitting logic.
